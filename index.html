Preguntas Frecuentes

¿Qué es el Sistema de Gestión de Energía (SEMS)?

+

SEMS es una aplicación web que te permite monitorear y gestionar el consumo de energía en tu hogar o negocio en tiempo real. Te proporciona reportes detallados y alertas para que puedas tomar decisiones informadas sobre tus hábitos de consumo.

¿Cómo me ayuda SEMS a ahorrar dinero?

+

Al identificar los dispositivos que consumen más energía, puedes ajustar tus hábitos de uso para reducir tu factura de electricidad. Los reportes y recomendaciones personalizadas te guían para optimizar tu consumo.

¿Necesito comprar hardware adicional para usar SEMS?

+

Sí, la aplicación SEMS funciona en conjunto con nuestros dispositivos inteligentes (como tomacorrientes y termas) que se conectan a tu red Wi-Fi para medir el consumo de energía y enviar los datos a la plataforma.

¿Ofrecen planes para empresas o solo para hogares?

+

Actualmente, nuestros productos y servicios están diseñados para usuarios domésticos. Estamos trabajando en soluciones más grandes para el futuro.
